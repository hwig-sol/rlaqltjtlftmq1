강사명

바이브코딩 워크숍 강사

  • 비개발자 출신, AI에게 말로 지시해서 서비스를 만듭니다
  • "개발을 모르는 사람의 눈높이"에서 가르칩니다
  • Claude Code로 업무 자동화 도구를 직접 제작·운영 중
바이브 코딩이란

개발만이 아닙니다

  • 📋
    문서/기획 — 사업 계획서, 시장 조사, 마케팅 자료, 구글드라이브 1800개 파일 정리 기획자·마케터가 AI에게 말로 지시해서 문서를 자동 생성
  • 🔧
    업무 관리 — 김비서 대시보드, 노션/구글 연동, 업무 자동화 반복 업무를 AI가 대신 처리하고, 대시보드로 한눈에 확인
  • 🌐
    사이트/도구 개발 — 웹사이트, 앱, 포트폴리오, 게임 데모 말로 지시하면 AI가 코드를 쓰고 결과물을 만들어줍니다

이 세 가지가 동등하게 "말로 지시할 수 있는 것들"입니다.
개발이 특별한 게 아니에요. 전부 같은 방식으로 만들 수 있습니다.

Claude Desktop 설치

오늘은 이것만 설치하면 됩니다

  • Claude 앱 다운로드 (claude.ai/download)
  • 설치 후 실행 → Claude Pro 계정으로 로그인
  • 오른쪽에 "코드" 탭 클릭 → 이 화면이 나오면 성공
  • 먼저 되신 분은 옆 사람 도와주세요

"이 사이트에 접속해서 구조, 디자인, 콘텐츠를 분석하고 HTML 보고서로 만들어줘"
  • 경쟁사 사이트 또는 자사 사이트 링크만 제공하면 됩니다
  • AI가 접속 → 분석 → 보고서 자동 생성
  • 결과물을 브라우저에서 바로 확인
