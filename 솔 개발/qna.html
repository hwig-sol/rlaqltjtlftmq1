❓ FAQ
💰 비용 관련
수강료가 얼마인가요? ▾
각 강의당 5만원입니다.
전체(1~4강) 수강 시 별도 패키지 할인이 있을 수 있으니 문의해 주세요.
Claude 유료 구독 비용은 별도인가요? ▾
네, Claude Pro 구독비 ($20/월)는 수강료와 별도입니다.
1강 전에 미리 가입하고 유료 구독을 완료해 오셔야 실습이 가능합니다.
환불이 가능한가요? ▾
환불 규정은 오픈카톡 또는 이메일(psol@hellovr.kr)로 문의해 주시면 안내드리겠습니다.
🎓 수업 관련
코딩을 전혀 몰라도 되나요? ▾
네, 전혀 몰라도 됩니다!
이 워크숍은 비개발자를 위해 설계되었습니다. 코드를 직접 쓰지 않고, AI에게 한국어로 말하는 방식으로 진행됩니다.
각 강의에서 구체적으로 뭘 만드나요? ▾
1강: Claude 앱으로 업무 보조 도구 만들기. 자료 조사, 웹사이트 분석, 데이터 정리 등 Claude의 주요 기능을 직접 써봅니다.

2강: 본인이 구상해 온 아이디어를 실제 웹페이지/서비스로 만들고, Vercel로 전 세계에 배포합니다. API 연결 전이라 "껍데기" 상태지만, 직접 인터랙티브한 결과물이 나옵니다.

3강: 2강에서 만든 서비스에 실제 API를 연동합니다. Gmail, Notion, AI 등 외부 서비스와 연결해서 진짜 작동하는 서비스로 만듭니다.

4강: 멀티 디바이스 개발, 협업 개발, 멀티 에이전트 활용법을 배웁니다. 터미널마다 역할(팀장, 백엔드, 프론트엔드, 디자이너)을 부여해서 동시 개발하는 방법도 배웁니다.
1강만 듣고 빠져도 되나요? ▾
네, 각 강의는 개별 수강이 가능합니다. 다만 2강부터는 이전 강의의 결과물을 이어서 발전시키기 때문에 순서대로 듣는 것을 권장합니다.
멀티 에이전트가 뭔가요? ▾
터미널(CMD)을 여러 개 열어서 각각에 서로 다른 역할을 부여하는 기법입니다.

예를 들어:
• 터미널 1 → "너는 프로젝트 팀장이야"
• 터미널 2 → "너는 백엔드 개발자야"
• 터미널 3 → "너는 UI 디자이너야"

이렇게 하면 혼자서도 팀 단위 개발을 시뮬레이션할 수 있습니다. 4강에서 직접 실습합니다.

💻 준비물
Windows와 Mac 중 뭐가 좋나요? ▾
둘 다 상관없습니다. Claude Code는 Windows, Mac 모두 지원합니다.
평소 사용하시는 노트북을 가져오시면 됩니다.
3강에서 해외 결제 카드가 꼭 필요한가요? ▾
API 연동 실습 시 일부 서비스(OpenAI 등)에서 해외 결제가 필요할 수 있습니다.
필수는 아니지만, 준비해 오시면 실습 범위가 넓어집니다.
💡 대부분의 API는 무료 크레딧이 있어서 실습 단계에서는 큰 비용이 발생하지 않습니다.
