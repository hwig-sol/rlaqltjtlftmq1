2. 핵심 UI 컴포넌트 비교

UI 요소 웅진씽크빅 한우리독서토론논술
GNB 로고 + 5개 텍스트 메뉴 + 검색 아이콘 + 햄버거. 우측 슬로건 텍스트 애니메이션 검색 있음 로고 + 4개 텍스트 메뉴 (가로 균등). 햄버거 메뉴로 전체 메뉴 확장 검색 없음
히어로 영역 풀와이드 슬라이더 6장, 이벤트/캠페인 중심. 하단에 페이지 인디케이터 + 진행 바 전환형 풀스크린 단일 섹션 + CSS 텍스트 로테이션({읽기} {쓰기} {말하기}). 강렬한 오렌지 배경 몰입형
탐색/필터 3단 드롭다운 필터 (학습유형, 나이, 과목) + 검색 버튼. 하단에 아이콘 퀵메뉴 8개 별도 필터/검색 없음. 메뉴 + 스크롤로만 탐색
콘텐츠 구조 탭 전환 (씽크빅/스마트올/북클럽) → 제품 카드 그리드. 데이터 중심 레이아웃 풀스크린 섹션 스크롤 (읽기→쓰기→말하기→실적→프로그램). 스토리텔링 중심
CTA (전환) 하단 고정바 "체험/상담 신청" + 우측 "1:1 문의" FAB + 상단 "스크롤 탑" FAB 하단 고정바 "상담 시 Npay 상품권" + 우측 플로팅 다중 팝업(교사모집, 설명회 등)
색상 체계 배경 #FFF, 포인트 오렌지 #FF9800, 텍스트 #333. 밝고 깔끔한 톤 배경 강한 오렌지-레드 #E8491D 풀섹션, 텍스트 #FFF. 대담하고 강렬한 톤
일러스트 실사 사진 + 제품 목업 (태블릿, 학습지) 중심 3D 렌더링 일러스트 (책, 연필, 말풍선) + 실사 모델 (선생님+학생) 혼합
반응형 완전 반응형. 모바일에서 하단 탭바 네비게이션 전환 반응형. 모바일에서 상단 고정 GNB 유지, 풀스크린 섹션 스크롤 그대로
핵심 차이: 전환 퍼널 vs 브랜드 스토리

웅진은 "검색 → 필터 → 제품 선택 → 체험 신청" 퍼널을 메인에 직접 배치하여 즉각적 전환을 유도합니다. 한우리는 "읽기-쓰기-말하기 철학 → 35년 신뢰 → 프로그램 안내 → 상담" 스토리를 따라가며 학부모의 공감을 이끌어냅니다. 비슷한 서비스를 만든다면, 이 두 접근의 하이브리드가 이상적입니다.

3. 사용자 여정(User Flow) 비교

웅진씽크빅 — 탐색 중심 플로우
  • 메인 진입 → 슬라이더 배너로 이벤트 노출
  • 3단 필터(유형/나이/과목)로 즉시 맞춤 검색
  • 퀵메뉴 8개 아이콘으로 주요 브랜드 원클릭 이동
  • 탭 UI로 3개 브랜드(씽크빅/스마트올/북클럽) 전환
  • 제품 카드 → 상세 페이지 → 체험/상담 신청 폼
  • 하단 고정 CTA 바 항상 노출 (이탈 방지)
한우리 — 스토리 중심 플로우
  • 메인 진입 → 읽기/쓰기/말하기 로테이션 애니메이션으로 핵심가치 전달
  • 스크롤 → 교육 철학 메시지 (감성 카피라이팅)
  • 3개 역량별 풀섹션 (읽기→쓰기→말하기) + 3D 일러스트 + 3단 프로세스
  • 35년 역사 + 실적 수치 (센터수, 교사수, 회원수)
  • 프로그램 카드 (정규/심화/특강 탭 전환)
  • 플로팅 상담 배너 + Npay 인센티브

4. 재사용 가능한 UI 컴포넌트 목록

컴포넌트 웅진에서 참고 한우리에서 참고
검색/필터 바 3단 드롭다운 + 검색 버튼 조합. 헤딩 문구("적합한 학습을 찾아보세요")로 맥락 제공 -
퀵메뉴 원형 아이콘 + 라벨 8개 가로 배치. 한 줄로 주요 카테고리 커버 -
프로그램 카드 썸네일 + 제목 + 설명, 그리드 배치 컬러풀 일러스트 카드, 탭(정규/심화/특강) 전환, 가로 스크롤 가능
풀스크린 섹션 - 배경색 전환 + 3D 일러스트 + 큰 타이포 + 3단 프로세스 설명. 모바일에서 매우 효과적
하단 고정 CTA "체험/상담 신청" 텍스트 + 동기부여 문구 조합 상품권 인센티브 강조 + Npay 뱃지. 모델 이미지 포함
탭 전환 라운드 필/고스트 버튼 탭 (브랜드 전환) 라운드 필 버튼 탭 (정규/심화/특강)
수치 카운터 - 전국 센터수 / 교사수 / 회원수 카운트업 애니메이션. 신뢰감 형성
플로팅 요소 우측 하단 "1:1 문의" + "TOP" FAB 2개 우측 다중 플로팅 팝업(교사모집, 설명회, 특별반 등) + "TOP" FAB

5. 내 서비스에 적용할 Takeaway

맞춤 검색 필터는 필수 연령/과목/유형 3단 필터를 메인 상단에 배치. 학부모가 원하는 프로그램에 3초 내 도달하게.
참고: 웅진 — 히어로 바로 아래 필터 바
교육 철학 스토리텔링 섹션 "왜 우리 서비스인가"를 풀스크린 + 큰 타이포 + 3D 일러스트로 임팩트 있게. 메인에 최소 1~2 섹션.
참고: 한우리 — 읽기/쓰기/말하기 풀섹션
퀵메뉴 아이콘 바 주요 서비스(과목별/연령별)를 아이콘+라벨 1줄로 배치. 모바일에서 가로 스크롤.
참고: 웅진 — 8개 원형 아이콘 퀵메뉴
수치 카운터로 신뢰 확보 회원수, 교사수, 만족도 등 핵심 KPI를 카운트업 애니메이션으로 노출.
참고: 한우리 — 센터수/교사수/회원수 영역
하단 고정 CTA + 인센티브 "무료 상담 신청" 고정바 + 상품권/할인 프로모션 뱃지 조합. 스크롤해도 항상 노출.
참고: 양쪽 모두 — 하단 고정 전환 바
프로그램 카드 + 탭 전환 정규/심화/특강 등 프로그램 유형을 탭으로 전환하고, 카드 형태로 시각화. 모바일에서 가로 스크롤.
참고: 한우리 — 3탭 프로그램 카드 섹션
색상 전략: 화이트+포인트 컬러 기본 배경은 화이트(웅진식 깔끔함), 핵심 섹션만 풀컬러 배경(한우리식 임팩트) 혼합.
참고: 하이브리드 전략
모바일: 하단 탭바 네비게이션 모바일에서는 상단 GNB를 간소화하고 하단 탭바(홈/프로그램/상담/마이)로 전환. 모바일 사용성 극대화.
참고: 웅진 — 모바일 하단 탭 네비게이션
종합: 이상적인 서비스 메인 페이지 구성 제안

GNB(로고+메뉴+검색) → 히어로(핵심 가치 스토리텔링 1장 + 프로모션 배너) → 맞춤 필터 바(연령/과목/유형) → 퀵메뉴 아이콘 → 교육 철학 풀섹션(3D 일러스트 + 큰 타이포) → 프로그램 카드(탭 전환) → 신뢰 지표(수치 카운터) → 후기/추천도서 → 하단 고정 CTA
